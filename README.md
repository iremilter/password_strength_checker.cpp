# Password Strength Checker in C++

This project is a simple C++ console program that checks the strength of a user-entered password.

## Description

The program asks the user to enter a password and analyzes it based on several criteria such as length, uppercase letters, lowercase letters, digits, and special characters. Based on these checks, the program classifies the password as **Weak**, **Medium**, or **Strong**.

## Features

* User password input
* Checks for:

  * Uppercase letters
  * Lowercase letters
  * Digits
  * Special characters
* Password strength evaluation
* Beginner-friendly C++ implementation

## Technologies Used

* C++
* Standard Library (`iostream`, `string`, `cctype`)

## Example

Input:
Abc123!

Output:
Password strength: Strong
